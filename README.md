# Power BI Dashboards
This repository contains interactive Power BI dashboards created for data analysis practice.

1. Brew Hub Sales Dashboard
   - Analyzes sales trends, product categories, and hourly performance.

2. Sales Performance Dashboard
   - Visualizes total sales, transactions, and product insights.

3. Airbnb Analysis Dashboard
   - Analyzes listing prices, locations, and booking trends.

Tools Used:
- Power BI
- Data Visualization
- Dashboard Design
